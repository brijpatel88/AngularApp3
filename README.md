# Angular Book Management App

## 📺 Project Demo (YouTube)
Watch the full project demo here:
https://youtu.be/5po8a8fKzms

## Features
- View book list
- View book details
- Add new book
- Update book
- Delete book

## Technologies
- Angular (Standalone Components)
- Angular Routing
- HttpClient
- In-Memory Web API

## How to Run
npm install
ng serve -o
